# 百世江湖 全球化定位分析

## 一、国际玩家核心画像

### 1.1 Steam全球玩家特征

| 维度 | 欧美玩家 | 日韩玩家 | 中国玩家 |
|------|---------|---------|---------|
| **主力平台** | PC + 主机 | 主机 + PC | 手机 + PC |
| **核心动机** | 世界探索、沉浸剧情、操作挑战 | 收集养成、剧情、角色魅力 | 社交竞技、数值成长、效率 |
| **付费偏好** | 买断制、订阅制 | 买断制 + 抽卡 | F2P + 内购 |
| **社交倾向** | 偏独处、深度单人体验 | 单人 + 轻社交 | 重社交、开黑 |
| **耐心阈值** | 中等（愿意学习复杂系统） | 高（接受慢热） | 低（3章定生死） |
| **美术偏好** | 风格化 > 写实 | 精致二次元 | 3A级画面 |

### 1.2 成功出海的中国游戏分析

| 游戏 | 全球成功原因 | 百世江湖可借鉴 |
|------|------------|-------------|
| **黑神话：悟空** | 顶级画面 + 中国神话 + 动作系统 | 中国题材可以打动全球，但需要顶级品质 |
| **原神** | 开放世界 + 角色魅力 + 跨平台 | 角色设计和世界观可以跨文化 |
| **暖雪** | Roguelike + 中式美学 + 爽快战斗 | 中式风格 + 循环机制 = 全球市场 |
| **部落与弯刀** | 沙盒 + 势力系统 + 自由度 | 系统深度可以吸引全球玩家 |
| **太吾绘卷** | 复杂系统 + 中国武侠 + 自由度 | 深度模拟 + 武侠 = 全球 niche 市场 |

|------|------|------------|

## 三、全球化设计调整

### 3.1 必须调整的设计

| 原设计 | 问题 | 全球化调整 |
|--------|------|-----------|
| 纯文字模拟 | 国际玩家需要视觉反馈 | 文字 + 简约图形（类似Cultist Simulator） |
| 中国武侠术语 | 翻译困难 | 保留核心术语 + 提供英文解释 |
| 社交开黑 | 国际玩家偏独处 | 单人为主，可选异步社交 |
| 数值成长 | 国际玩家反感P2W | 成长基于选择，不是氪金 |
| F2P模式 | 国际核心玩家排斥 | 买断制（$15-25） |

### 3.2 必须保留的设计

| 设计 | 全球价值 | 原因 |
|------|---------|------|
| 转生系统 | ★★★★★ | Roguelike循环全球热门 |
| 记忆系统 | ★★★★★ | 前所未见的创新 |
| NPC关系网 | ★★★★☆ | 深度模拟的核心 |
| 世界历史 | ★★★★☆ | 涌现叙事的基础 |
| 选择后果 | ★★★★★ | 叙事RPG的核心 |

### 3.3 全球化MVP调整

**旧MVP（中国优先）**:
```
文字版江湖模拟器 → 记忆系统 → 转生系统 → NPC AI
```

**新MVP（全球优先）**:
```
Roguelike循环原型（转生 + 属性继承）
    ↓
记忆系统（NPC记住前世）
    ↓
简约视觉（Cultist Simulator风格）
    ↓
英文本地化
    ↓
Steam Next Fest测试
```

## 六、最终定位

### 一句话定位（全球版）

> **"A reincarnation simulation where the world remembers every life you've ever lived."**
> 
> "一个轮回模拟器，世界记住你活过的每一世。"

### Steam标签（全球版）

- Roguelike
- Simulation
- Narrative
- choices matter
- multiple endings
- dark fantasy
- medieval
- atmospheric
- indie

### 核心卖点排序（全球版）

1. **World Remembers** — 世界记住你的每一世（创新点）
2. **Reincarnation Loop** — 轮回循环（Roguelike元素）
3. **Emergent Stories** — 涌现叙事（每个玩家的故事都不同）
4. **Deep Systems** — 深度模拟（RimWorld级别的系统深度）
5. **Jianghu Aesthetic** — 武侠美学（差异化）
