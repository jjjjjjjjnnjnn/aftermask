# 百世江湖 游戏设计圣经

## 一句话描述

> 一个会记住你百世轮回的武侠世界。你每次死亡都会转生，世界持续演化，NPC记住你的历史，你创造自己的传奇。

## 核心体验

### 玩家情感旅程
```
第一世: 茫然 → 探索 → 成长 → 死亡 → 惊讶(转生了!)
第二世: 理解 → 策略 → 崛起 → 被背叛 → 愤怒
第三世: 复仇 → 权谋 → 称霸 → 寿终正寝 → 成就感
第N世: 传奇 → 世界记住你 → 新玩法解锁 → ...
```

### 核心循环
```
出生(随机属性/背景)
    ↓
学习/修炼(获得技能)
    ↓
冒险/社交(建立关系)
    ↓
冲突/选择(塑造命运)
    ↓
死亡/寿终(总结一世)
    ↓
转生(继承/遗忘/变异)
    ↓
(循环)
```

## 世界观

### 时代背景
架空武侠世界"百世界"。这个世界有一个特殊法则：灵魂不灭，死后转生。但转生不是简单的重生——你会丢失大部分记忆，只保留模糊的直觉和少数关键记忆碎片。

### 地理（MVP）
- **落霞村**: 新手村，平静的农村
- **醉仙楼客栈**: 情报中心，江湖消息汇集地
- **铁掌门武馆**: 基础武学教学
- **野猪林**: 危险区域，有低级野兽
- **官道**: 连接各地的主要道路

### 势力（MVP）
- **铁掌门**: 正派武馆，重视武德
- **毒蛇帮**: 江湖帮派，利益至上
- **官府**: 维持秩序，但腐败
- **散人**: 无门无派的江湖人

## 角色系统

### 属性
| 属性 | 作用 | 影响 |
|------|------|------|
| 力量 | 物理攻击、负重 | 近战伤害、搬运能力 |
| 敏捷 | 闪避、速度、暗器 | 先手率、逃跑成功率 |
| 悟性 | 学习速度、领悟 | 技能获取速度、对话选项 |
| 魅力 | 社交、说服、威慑 | NPC好感度、交易价格 |

### 境界（简化版）
```
乞丐 → 农民 → 学徒 → 武者 → 武师 → 武宗 → 武圣 → 武神
```

每个境界分1-9层。突破需要：修为 + 领悟 + 机缘。

### 职业（MVP）
| 职业 | 起始属性 | 特点 |
|------|----------|------|
| 乞丐 | 力2敏3悟4魅5 | 社交强，学习快，初始穷 |
| 农民 | 力5敏3悟3魅3 | 体力好，基础扎实 |
| 学徒 | 力3敏3悟5魅3 | 悟性高，适合修炼 |

## 转生系统

### 转生规则
1. **死亡触发转生** — 自然死亡或意外死亡
2. **记忆丢失** — 大部分记忆消散，保留1-3个"前世记忆碎片"
3. **属性继承** — 少量属性加成（基于前世成就）
4. **世界记忆** — NPC和世界记住你前世的行为
5. **声望传承** — 前世的名声会影响下一世NPC的态度

### 前世记忆碎片
- 通关条件：前世达到特定成就
- 效果：解锁特殊对话选项、隐藏地点、特殊技能
- 数量限制：最多保留3个

### 轮回奖励
| 一世成就 | 轮回奖励 |
|----------|----------|
| 修炼到武师 | 下一世+2悟性 |
| 建立一个势力 | 下一世解锁"领袖"天赋 |
| 帮助10个人 | 下一世NPC初始好感+10 |
| 杀害无辜 | 下一世NPC初始恐惧+20 |
| 寿终正寝 | 下一世+5年寿命 |

## NPC系统

### NPC属性
每个NPC拥有：
- **人格**: 5维度性格模型（开放性/尽责性/外向性/宜人性/神经质）
- **关系**: 与其他NPC和玩家的关系值
- **目标**: 当前追求的目标
- **记忆**: 对玩家行为的记忆（正面/负面）
- **状态**: 当前行为状态

### NPC记忆
- **短期记忆**: 最近7天的事件
- **长期记忆**: 重要事件（玩家行为、重大冲突）
- **传说记忆**: 跨世代的记忆（通过口口相传）

### NPC行为
- **日常循环**: 起床→工作→社交→休息
- **关系驱动**: 根据关系值调整行为
- **目标驱动**: 根据当前目标行动
- **记忆驱动**: 根据记忆调整态度

## 战斗系统（MVP简化版）

### 回合制
```
速度决定行动顺序
    ↓
选择行动(攻击/防御/技能/逃跑)
    ↓
结算伤害
    ↓
检查胜负
    ↓
下一回合
```

### 战斗属性
- 攻击力 = 力量 × 武器系数
- 防御力 = 力量 × 0.5 + 装备
- 闪避率 = 敏捷 × 0.1
- 暴击率 = 敏捷 × 0.05 + 悟性 × 0.02

## 经济系统（MVP）

### 货币
- **铜钱**: 基础货币
- **银两**: 高级货币（100铜 = 1银）
- **金锭**: 稀有货币（100银 = 1金）

### 交易
- NPC有各自的商店
- 价格受关系值影响
- 好感度高 → 折扣
- 好感度低 → 涨价或拒绝交易

## UI设计原则

### 信息层级
```
第一层: 关键信息（血量、当前目标）
第二层: 重要信息（属性、装备）
第三层: 详细信息（技能、关系、历史）
```

### 风格
- 武侠水墨风格
- 简洁不花哨
- 信息密度适中
- 支持手柄操作

## 传播性设计

### 分享点
- "我活了100世" — 轮回次数统计
- "NPC居然记得我" — 世界记忆截图
- "我建立了一个门派" — 势力成就
- "我被整个江湖追杀" — 声望系统

### 成就系统
- 一世成佛：不杀一人通关
- 血腥之路：杀100人
- 百世轮回：完成100次转生
- 传奇人物：被10个NPC记住名字
