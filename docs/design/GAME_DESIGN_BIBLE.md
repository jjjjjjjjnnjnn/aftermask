# 百世江湖 游戏设计圣经 v2

## 一句话描述

> Build a legacy that survives your death.

每一世都会改变未来的世界。你不是因为「被记住」而继续轮回，而是因为「改变世界」而继续轮回。

## 核心体验

### 玩家情感旅程

```
第一世: 探索 → 成长 → 创造遗产 → 死亡 → 遗产开始变化
第二世: 见证后果 → 好奇 → 干预 → 创造新遗产 → 死亡 → 世界再次改变
第三世: 世界已经完全不同 → 传奇 → 想看看最终会怎样
第N世: 你创造的历史成为传说 → 你改变了整个世界
```

### 核心循环

```
出生(随机属性/背景)
    ↓
成长/修炼(获得技能)
    ↓
行动/选择(创造遗产)
    ↓
死亡(遗产被世界继承)
    ↓
转生(见证遗产变化)
    ↓
再次行动(改变遗产未来)
    ↓
(循环)
```

### 核心卖点排序

1. **Legacy** — 你的行为永久改变世界
2. **Consequences** — 每个选择都有跨世后果
3. **Emergent Stories** — 每个玩家的故事都不同
4. **Deep Systems** — RimWorld级别的系统深度
5. **Jianghu Aesthetic** — 东方美学差异化

## 世界观

### 时代背景

架空东方世界「百世界」。这个世界有一个特殊法则：灵魂不灭，死后转生。但转生不是简单的重生——你会丢失大部分记忆，只保留模糊的直觉和少数关键记忆碎片。

**关键区别**：你不是因为「被记住」而重要，而是因为「你改变了世界」而重要。

### 地理（MVP）

- **落霞村**: 新手村，平静的农村
- **醉仙楼客栈**: 情报中心，江湖消息汇集地
- **铁掌门武馆**: 基础武学教学
- **野猪林**: 危险区域，有低级野兽
- **官道**: 连接各地的主要道路

### 势力（MVP）

- **铁掌门**: 正派武馆，重视武德
- **毒蛇帮**: 江湖帮派，利益至上
- **官府**: 维持秩序，但腐败
- **散人**: 无门无派的江湖人

## Legacy系统

### 四个维度

#### 1. 物理遗产 (Physical Legacy)
玩家行为改变的物理世界状态
- 你建造的建筑
- 你种的树
- 你留下的物品
- 你破坏的地方

#### 2. 社会遗产 (Social Legacy)
玩家行为改变的势力格局
- 你创建的门派
- 你调解的纷争
- 你背叛的盟友
- 你建立的学校

#### 3. 传说遗产 (Legend Legacy)
玩家行为创造的故事
- 说书人讲你的故事
- 你留的信百年后被发现
- 你创造的节日还在过
- 你的善行被传颂

#### 4. 血脉遗产 (Blood Legacy)
玩家行为影响的后代
- 你的孩子传承祖训
- 你抛弃的孩子的后代要复仇
- 你不知道的血脉

### Legacy动态变化

遗产不是静态的。它会随时间变化：

| 机制 | 说明 | 示例 |
|------|------|------|
| 遗产失控 | 你创建的东西偏离初衷 | 你创建的门派变成邪教 |
| 历史误解 | 后人误解你的行为 | 你救了人，史书记载你杀了人 |
| 后代分化 | 你的后代走向不同道路 | 你的儿子成为英雄，孙子成为恶棍 |
| 传说扭曲 | 你的故事被改编 | 你救人的故事变成你杀人的传说 |

## 转生系统

### 转生规则

1. **死亡触发转生** — 自然死亡或意外死亡
2. **记忆丢失** — 大部分记忆消散，保留1-3个"前世记忆碎片"
3. **属性继承** — 少量属性加成（基于前世成就）
4. **遗产继承** — 你的行为永久改变世界
5. **后果见证** — 你会看到你的行为造成了什么后果

### 前世记忆碎片

- 通关条件：前世达到特定成就
- 效果：解锁特殊对话选项、隐藏地点、特殊技能
- 数量限制：最多保留3个

### 轮回奖励

| 一世成就 | 轮回奖励 |
|----------|----------|
| 修炼到武师 | 下一世+2悟性 |
| 建立一个势力 | 下一世解锁"领袖"天赋 |
| 帮助10个人 | 下一世NPC初始好感+10 |
| 杀害无辜 | 下一世NPC初始恐惧+20 |
| 寿终正寝 | 下一世+5年寿命 |

## NPC系统

### NPC属性

每个NPC拥有：
- **人格**: 五维度性格模型
- **关系**: 与其他NPC和玩家的关系值
- **目标**: 当前追求的目标
- **记忆**: 对玩家行为的记忆
- **状态**: 当前行为状态
- **Legacy反应**: 对玩家遗产的态度

### NPC记忆

- **短期记忆**: 最近7天的事件
- **长期记忆**: 重要事件
- **传说记忆**: 跨世代的记忆
- **Legacy感知**: 对玩家遗产的反应
