# Legacy System v0 架构设计

## 核心命题

> **玩家不是因为「被记住」而继续轮回，而是因为「改变世界」而继续轮回。**

旧命题：The world remembers every life.
新命题：**Every life leaves a mark on the world.**

## Legacy 的定义

Legacy = 玩家在前世的行为对世界造成的**不可逆改变**。

关键区别：
- 记忆 = NPC记得你（被动）
- 遗产 = 世界因你而不同（主动）

## Legacy 的四个维度

### 1. 物理遗产 (Physical Legacy)
玩家行为改变的物理世界状态

### 2. 社会遗产 (Social Legacy)
玩家行为改变的势力格局和人际关系

### 3. 传说遗产 (Legend Legacy)
玩家行为创造的传说和故事

### 4. 血脉遗产 (Blood Legacy)
玩家行为影响的后代和传承

## 20个具体案例

### 物理遗产案例

#### 案例1：你救了村子 → 村子为你立碑

**第一世**：你击退了入侵的匪帮，救了落霞村。

**第二世结果**：
```
落霞村中央有一座石碑。

碑文：
「百年前，有一人独退三十匪。
 村人感其恩德，立碑以记。
 碑上无名，村人代代口传。」

你走近石碑，村民说：
「这碑每年清明都有人来打扫。
 我爷爷的爷爷说，那是我们村的恩人。」
```

**影响**：
- 落霞村对你的初始好感 +30
- 你在这个村子更容易获得信任
- 隐藏事件触发：石碑下的宝箱（你前世埋的）

---

#### 案例2：你烧了寺庙 → 永久废墟

**第一世**：你为了报复，烧毁了铁掌门的藏经阁。

**第二世结果**：
```
铁掌门的后山有一片焦黑的废墟。

杂草丛生，无人修缮。

新掌门说：
「祖师曾留下遗训：藏经阁被毁之日，
 是铁掌门衰落之始。后人当铭记，
 不可轻信外人。」

你走进废墟，发现地下密室还在。
 里面有一本残破的秘籍——你前世没能带走的东西。
```

**影响**：
- 铁掌门对你的初始恐惧 +20
- 门派实力永久削弱（秘籍失传）
- 隐藏事件触发：地下密室（你前世留下的痕迹）


**影响**：
- 落霞村对你的初始好感 +10
- 特殊地点解锁：柳树下（前世记忆碎片触发点）
- 情感触发：你和这个世界有连接

### 社会遗产案例

#### 案例6：你创建了门派 → 门派兴衰

**第一世**：你创建了「青云门」，发展到鼎盛时期。

**第二世结果**：
```

掌门对新弟子说：
「这是我们门派的祖师。
 他创立了青云门，定下了三条门规：
 一、不可欺压弱者；
 二、不可背叛同门；
 三、不可忘记本心。」

你站在门外听着，心中五味杂陈。
```

**影响**：
- 青云门对你的初始好感 +50（如果你是好人）
- 门派声望系统：你的行为影响门派当前状态
- 特殊事件触发：祖师画像的真相

#### 案例8：你调解了纷争 → 和平百年

## Legacy 系统架构

### Legacy 数据结构

```gdscript
class_name LegacyRecord
extends Resource

@export var source_era: int          # 来源世代
@export var source_entity_id: int    # 来源实体
@export var legacy_type: String      # 物理/社会/传说/血脉
@export var target_location: String  # 影响地点
@export var target_faction: String   # 影响势力
@export var target_entity: int       # 影响实体
@export var impact_value: int        # 影响值 (-100 ~ +100)
@export var is_permanent: bool       # 是否永久
@export var description: String      # 描述
@export var trigger_conditions: Array # 触发条件
```

### Legacy 生成流程

```
玩家行为
    ↓
判断是否产生Legacy
    ↓
┌─────────────────────────────────────┐
│ Legacy类型判断：                     │
│                                     │
│ 物理遗产：建筑/破坏/留下物品         │
│ 社会遗产：创建势力/背叛/调解         │
│ 传说遗产：救人/杀人/创造故事         │
│ 血脉遗产：生育/抛弃/收养             │
└─────────────────────────────────────┘
    ↓
生成Legacy记录
    ↓
写入WorldLegacy数据库
    ↓
下一世加载Legacy
    ↓
影响NPC态度/世界状态/事件触发
```

### Legacy 影响机制

```
Legacy影响 = impact_value × decay_rate × era_gap

其中：
- impact_value: -100 ~ +100
- decay_rate: 0.95/世（每世衰减5%）
- era_gap: 距离前世的世代数

示例：
- impact_value = +80（救了村子）
- decay_rate = 0.95
- era_gap = 3（过了3世）

Legacy影响 = 80 × 0.95^3 = 68.6

村民对你的好感 +68.6（仍然很高）
```

### Legacy 触发条件

| Legacy类型 | 触发条件 | 影响范围 |
|-----------|---------|---------|
| 物理遗产 | 你建造/破坏/留下的物理存在 | 地点 |
| 社会遗产 | 你创建/摧毁/改变的势力 | 势力 |
| 传说遗产 | 你创造的传说和故事 | 区域 |
| 血脉遗产 | 你的后代和血缘关系 | 个人 |

### Legacy 优先级

| 优先级 | Legacy类型 | 玩家感知 |
|--------|-----------|---------|
| P0 | 物理遗产（你留下的东西） | 最强 |
| P0 | 社会遗产（你改变的势力） | 很强 |
| P1 | 传说遗产（你创造的故事） | 中等 |
| P2 | 血脉遗产（你的后代） | 情感 |
